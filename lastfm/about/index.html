NekoFM aggregates the lastfm usernames of people on nekoweb (sometimes even those not on nekoweb) and displays information about them such as the current playing song and if offline they are offline, the last played song, it is possible thanks to Tepiloxtl's Lastfm Websocket API


The site has two versions

  • Stable branch at /lastfm/

    Currently the only branch of lastfm is the stable branch. The beta branch is discontinued.

  • Beta branch at /lastfm/beta/

    The beta branch uses the /ws-bleeding/ endpoint on the websocket API which supplies a slightly different json payload and more features such as an nsfw boolean and cleaner json.
